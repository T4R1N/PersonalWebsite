RGB0

RGB0 is the most influential project I've ever undertaken.

It started out as a game I was inspired to make for fun, but turned into a competition-enterer for my school.

It's a platformer but I was concious of how games are supposed to make people feel. I knew how to program well and draw up good concepts for games.

It features a jetpack and a large focus on a bullet-hell aspect of play.

RGB0 takes a huge amount of inspiration from my older projects, even going as far as to take direct characters. It features a prison setup (PNB) and the icon boss is Deromn (Phase Trial).

I love this project because it consumed my life and I really enjoyed it. If you knew me, you knew about that game. I learned a lot while making it and I have a lot to show for it.

Celly mk-IV

Celly mk-IV is a discord bot that I made using Python.

The history behind the identity of the bot is that my dog's name is Celly. I made two bots (MK-II and MK-3) to be named after her. Neither were made with any programming language.

The reason that the icon is a hue-shifted Terminator is because I made the robot become progressively more evil-looking over the course of its marks. Celly, my dog, is adorable and fluffy. MK-II is very strange and unnverving. MK-3 is strange and unnerving in the sense that looking at a skinless human is unnerving. The Terminator is just outright awesome/badaxe to look at.

Celly mk-IV runs off of my Raspberry Pi 4. It can only be invited to servers by me.

It has lots of features. You can use 'hc' or 'hey celly' to start a command. You can then say 'rickroll' or 'don't' rickroll, and neither will rickroll you. If you say 'fart' in any way shape or form, the bot will subtract from the Puffhyu score. My friend's online alias is Puffhyu and I made a score based on how much people like him or not. You can say 'I love puffhyu' or 'I hate puffhyu' to raise or decrease the score by one, and that is registered into a text file that is saved locally.

If you ping it, it'll just tell you how to ask for commands. If you ping me/my account, it will say 'Tarin probably doesn't want to talk to you'. I thought that would be funny. My friend's bot says 'I will beat you to death' so mine says 'I promise not to beat you to death'. Unfortunately, I stopped adding to it because I felt like it wasn't really going anywhere, and I'm happy with how it turned out.

The github page is here.

Overgrowth

Overgrowth is a game I made in 7th grade (2019).

It was never taken past prototype but I think it's important to put it on here, because it's one of the first things I think of when I think of my historic games.

It is a plant-themed tower defense game where, for every enemy that you kill, you gain points. You could then either invest this into towers or you could level up the game.

Leveling up the game, depending on what level, can unlock towers as well as new enemies. For example, you unlock Buckets on level 5, which are indestructible to leaves, but can be killed by spores, which are unlocked on level 6.

Personally, I think it's okay in execution, but it's regrettable that I wasn't skilled enough and didn't have enoug motivation to let it come into its fullest fruition.

Video and playable link are coming soon.

Phase Trial

When I think of this game, I think of it as the successor to PNB.

It is another platformer with many similarities the rest of the ones that I made.

There is a lot of undescribable story, both laden within the game and in the making of it.

For example, the main character of the game is a green square named Tungsten. He is named this, but it never shows up in the actual game. Even though the only remnants of the game don't feature the other levels that I created, there is a lot of content that influences my game-making work to this day. I name every player layer 'Tung' and any time I add a story mode to games, it always has a sewer level (there is no way for me to recover it, sadly). Another example of this phenomenom is the character Deromn and what is associated with him. In Phase Trial, he represents how I used to talk at the time which I made him. Now, there is a boss in RGB0 which uses his face and name but talks differently. However, it uses the same soundtrack.

The sewer level featured slugs which came in all different sizes. It was meant to be gross, as the color scheme was brownish-yellow in nature.

Bubba Tech Utilities

This is a utility application that I made in C# with my dad.

Its primary focus was to have buttons to open zoom links for online learning, as it was tedious to have them bookmarked. Plus, I could customize it much better.

It also did other things such as have games that you can open.

It was a fun project, but I stopped using it after the school year ended. It was also tedious to have to input every new game I was interested in.

I was going to make it easy to change the links without having to go into the source code for profit, but I never got around to it. Maybe I will upload this to github eventually, but there's not a lot to see.
